## 1. 입력 채널 (3가지)

### 1.1 AI 대화형 입력
**위치:** 홈 화면 하단 고정 바 + 모든 화면 플로팅 버튼

**흐름:**
```
사용자 입력 → Claude API 파싱 → 미리보기 카드 표시 → 사용자 확인 → Firestore 저장
```

**예시 인풋:**
```
"우리집 인터넷 KT인데 월 33,000원 나가"
"정수기 코웨이 한달에 29,900원이고 내년 10월에 끝나"
"SKT 5G 스탠다드 55,000원 BC카드로 매월 20일에 나가. 약정 2025년 11월 종료"
"넷플릭스 유튜브 프리미엄 두개 쓰고 있어"
```

**AI 후속 질문 처리:**
- 누락 필드가 있으면 AI가 질문 생성
- 예: "약정 종료일도 알고 계신가요?" → 사용자 답변 → 항목 보완

**파싱 결과 미리보기 UI:**
```
┌─────────────────────────────────────┐
│ 📱 KT 인터넷           [통신]       │
│ 매월 25일 · 33,000원               │
│ 약정 만료: 2026.03 ⚠️              │
│ [수정] [추가하기]                   │
└─────────────────────────────────────┘
```

---

### 1.2 이미지/문서 업로드 입력
**지원 파일:** JPG, PNG, PDF, HEIC

**처리 대상:**
- 계약서 (약정 종료일, 금액, 약정 기간 추출)
- 고지서 / 청구서 (납부 금액, 납부일, 공급자 추출)
- 영수증 (금액, 날짜, 상호 추출)
- 문자 캡처 (결제 알림, 청구 안내 추출)
- 보험 증권 (보험사, 보장 내용, 갱신일 추출)

**Claude Vision API 프롬프트 전략:**
```
이미지에서 다음 정보를 추출하세요:
1. 문서 유형 (계약서/고지서/영수증/문자/증권)
2. 공급업체명
3. 금액 (원화)
4. 납부/결제일
5. 계약/약정 종료일
6. 고객번호 또는 계약번호
7. 납부 수단

불확실한 필드는 null로. 이미지가 관련 없으면 { "type": "unrelated" } 반환.
```

**UI 흐름:**
```
[카메라/갤러리 선택] → 이미지 미리보기 → [AI 분석 중...] → 파싱 결과 표시 → 사용자 확인/수정 → 저장
```

---

### 1.3 완전 수동 입력
**폼 필드:**
```
필수:
- 항목명 *
- 카테고리 * (드롭다운)
- 금액 * (숫자)
- 결제 주기 *

선택:
- 서브카테고리
- 공급업체명
- 납부일 (매월 N일)
- 결제 카드 (등록 카드 선택 또는 직접 입력)
- 계약 시작일
- 약정/계약 종료일 ← ⭐ 핵심 필드
- 자동갱신 여부 (토글)
- 해지 통보 필요 기간
- 무료체험 종료일
- 명의자
- 메모
- 첨부파일
```

## 2. 메인 대시보드 (`/`)

### 2.1 헤더 인사 영역
```typescript
// 시간대별 인사
const greeting = hour < 12 ? '좋은 아침이에요' : hour < 18 ? '안녕하세요' : '좋은 저녁이에요'

// 상태 메시지 (조건부)
const statusMsg =
  urgentCount > 0
    ? `⚠️ ${urgentCount}건이 긴급 처리가 필요해요`
    : expiringCount > 0
    ? `📋 ${expiringCount}건이 곧 만료돼요`
    : `✅ 오늘은 긴급한 항목이 없어요`
```

### 2.2 메트릭 카드 (4개)
| 카드 | 데이터 | 액센트 |
|------|--------|--------|
| 이번달 지출 예정 | 이번달 납부 예정 총액 | 기본 |
| 이번달 납부 건수 | 남은 납부 항목 수 | 기본 |
| 만료 임박 | 90일 내 계약 종료 건수 | warning |
| 카드 혜택 달성 | 최고 달성률 카드 % | success/warning |

### 2.3 이번달 납부 타임라인
```
오늘 (6월 7일)
│
├── D+3  [구독] 유튜브 프리미엄    14,900원  삼성카드
├── D+8  [통신] SK텔레콤           55,000원  BC카드
├── D+13 [렌탈] 정수기 코웨이      29,900원  자동이체
├── D+18 [공과금] 아파트 관리비    82,000원  자동이체
└── D+24 [통신] KT 인터넷          33,000원  신한체크
```

### 2.4 카드 혜택 현황 패널
```

[BC 바로카드]       ██████████  310,000 / 250,000원  ✅ 달성!
→ 캐시백 5,000원 예정
```

### 2.5 카테고리 요약 그리드

## 4. 카테고리 대시보드 (`/categories/[slug]`)

### 공통 구조
모든 카테고리 페이지가 동일한 레이아웃을 공유하며, 데이터만 다르다.

```
1. 카테고리 헤더
   - 아이콘, 이름, 색상 배경
   - 총 항목 수, 월 지출 합계, 연간 지출 합계

2. 요약 메트릭 (카테고리 맞춤)
   예: 보험 → "총 보장금액", "가장 빠른 갱신일"
   예: 구독 → "이번달 구독료", "무료체험 종료 임박"
   예: 세금 → "미납 세금", "다음 납부 기한"

3. 서브카테고리 필터
   예: 보험 → [전체] [실손] [자동차] [생명] [치아]

4. 항목 목록

5. 이 카테고리 인사이트 (있을 때)
   예: "같은 OTT 서비스를 2개 이상 구독 중이에요"
```

### 보험 카테고리 특화 기능
```
보험 포트폴리오 요약:
├── 실손보험 1건 (갱신일: 2025.12)
├── 자동차보험 1건 (갱신일: 2025.12.10) ⚠️ 128일 후
├── 종신보험 1건
└── 치아보험 1건

월 보험료 총계: 112,000원
연간 보험료: 1,924,000원
```

## 5. 카드 관리 (`/cards`)

### 5.1 카드 목록
- 등록 카드 카드뷰 (실제 신용카드처럼 시각화)
- 카드별 이번달 실적 한눈에

### 5.2 카드 등록 폼
```
카드명 *
발급사 * (신한, 삼성, 현대, KB, 롯데, 우리, 하나, BC...)
카드 종류 (신용/체크)
끝 4자리
카드 색상 (UI 표시용)

혜택 추가 (반복 추가 가능):
  - 혜택 설명 *
  - 조건 유형: 월 실적 / 카테고리별 실적 / 특정 가맹점
  - 조건 금액 (예: 300,000원 이상)
  - 할인 금액 또는 할인율
  - 할인 한도
  - 적용 카테고리 또는 가맹점
```

### 5.3 카드 상세 (`/cards/[id]`)

#### 실적 현황
```
이번달 실적: 248,000원 / 300,000원 목표
[████████░░░░] 82%

혜택 달성:
□ 통신비 10,000원 할인       52,000원 더 필요
✅ 카페 5% 적립               달성!
□ 주유 리터당 100원 할인      실적 부족
```

#### 이 카드로 결제 중인 항목
```
SK텔레콤        55,000원/월
KT 인터넷       33,000원/월
─────────────────────────
합계            88,000원/월 (이 카드 실적 기여)
```

#### 절약 시뮬레이션
```
💡 넷플릭스 (현재: 삼성카드)를 이 카드로 옮기면
   → 월 실적 306,000원 달성
   → 통신비 할인 10,000원 추가 획득
```

## 6. 해지·만료 관리 (`/expiry`)

이 화면은 **행동 유도(CTA)**가 핵심이다. 단순 목록이 아니라 "지금 뭘 해야 하는가"를 알려준다.

### 섹션 구조
```
🔴 즉시 처리 필요 (7일 이내)
─────────────────────────────
 SK텔레콤 약정        D-3    [지금 재약정하기 →]
 자동차보험 갱신      D-7    [비교견적 받기 →]

🟡 이번달 처리 예정 (30일 이내)
─────────────────────────────
 KT 인터넷 약정       D-18   [해지 / 재약정]
 정수기 코웨이 렌탈   D-25   [연장 / 해지]

🟢 3개월 이내
─────────────────────────────
 실손보험 갱신        D-67
 도메인 갱신          D-82

⚠️ 자동갱신 주의
─────────────────────────────
(autoRenews: true이고 계약 종료가 90일 이내인 항목)
 유튜브 프리미엄      D-12 갱신   [해지 설정 →]

🔘 이미 만료됨
─────────────────────────────
 SK텔레콤 (구) 약정   만료됨      [보관 / 삭제]
```

### 알림 설정
각 항목마다 알림 트리거 설정 가능:
- 만료 90일 전
- 만료 30일 전
- 만료 7일 전
- 만료 당일
- 해지 통보 마감일 기준 역산

## 8. AI 채팅 인터페이스

**위치:** 모든 화면에서 접근 가능 (하단 바 AI 버튼)

**지원 쿼리 유형:**

| 쿼리 유형 | 예시 |
|----------|------|
| 조회 | "SKT 약정 언제 끝나?" |
| 요약 | "이번달 돈 나갈 거 정리해줘" |
| 필터링 | "BC카드로 결제 중인 것들 보여줘" |
| 분석 | "제일 비싼 구독이 뭐야?" |
| 입력 | "넷플릭스 17,000원 매월 15일 삼성카드로 추가해줘" |
| 알림 설정 | "자동차보험 갱신 30일 전에 알려줘" |
| 절약 | "매달 지출 줄일 수 있는 거 뭐 있어?" |

**AI 응답 형식:**
- 단순 조회 → 텍스트 + 카드 컴포넌트
- 목록 → 미니 리스트
- 항목 추가/수정 → 확인 UI
